Inicializa o timer 0. E chamada durante a funcao main().

TMR0 = 39
Frequencia de 4 MHz, entao 1 us e o ciclo de instrucao. Para gerar um atraso de 1 segundo, sao necessarias 1.000.000 de ciclos. Como o prescaler esta setado para 256, entao Freq Timer 0 = 1M Hz / 256 = 3906,25 Hz. Periodo do Timer 0 = 1 / 3906,25Hz = 0,000256s ou 0,256ms ou 256us Fazendo TMR0 = 39, 256-39 = 217. 0,000256s x 217 = 0,055552s. 0,05555 x 18 = 0,999936s. 3906,25/217 = 18,00115. Temos que ter 18 interrupcoes para chegar a 1 segundo.
